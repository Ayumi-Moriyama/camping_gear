# キャンプギアのリスト＆重量計算アプリ(機能追加)

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - 自分が持っているキャンプギアの情報をデータベースに登録します。
  - 更新・削除機能を追加しました。
  - 登録したデータを表示し、選択したアイテムの重量の合計を計算します。（実装中）

## 工夫した点

  - JavaScriptでテーブルを作りましたがPHPで表示に変更しました。
  - テーブルの表示項目を増やしました。
  - チェックボックスを追加し、チェックしたアイテムの重量を合計したいです。（実装中）
