# キャンプギアのリスト＆重量計算アプリ

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - 自分が持っているキャンプギアの情報をデータベースに登録します。

  - 登録したデータを表示し、選択したアイテムの重量の合計を計算します。（実装中）

## 工夫した点

  - キャンプギアのサイズ表記を調べたら単位や書き方がバラバラだったので、単位を統一しました。
  - 小数点以下も表示できるように、データ型を調べました。
